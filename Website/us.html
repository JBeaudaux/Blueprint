The Blue print project needs you!

Blue print is an open project, that solely relies on the contribution and participation of volunteers to design, build and validate its solutions. In this context, the Blue print project is always looking for highly motivated contributors in the areas of:

  • # Mobile applications development: Android (Java, Py4A, SL4A) & iOS (Cococa)
  • # Embedded systems programming: C, RTOS, Contiki OS, Linux, Arduino
  • # Real-time data analysis: Statistics (descriptive, inferencial), pandas, NumPy, R
  • # Software engineering: Linux, Raspberry py, python, GUIs
