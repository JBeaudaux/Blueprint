Le projet Blue print a besoin de vous!

Blue print est un projet ouvert, se basant uniquement sur la contribution et la participation de volontaires pour concevoir, construire et valider ses solutions. Dans ce contexte, le projet Blue print est toujours à la recherche de contributeurs motivés dans des domaines tels que :

  • # Développement mobile : Android (Java, Py4A, SL4A) & iOS (Cococa)
  • # Programmation systèmes embarqués: C, RTOS, Contiki OS, Linux, Arduino
  • # Analyse de données en temps réel : Statistiques (descriptive, inférencielle), pandas, NumPy, R
  • # Ingéniérie logicielle: Linux, Raspberry py, python, GUIs
